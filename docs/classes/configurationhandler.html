Static Private setPropertyRec

  • setPropertyRec(config: any, key: string, value: any): any
    • Defined in src/Configuration.ts:135

    Helper function realizing the setProperty method. This method recursivly travers the current configuration and updates every occurence in the path with its new value. If last layer of the given key is reached, the config will be updated with the new value and the updated object will be returned. For any other layer the object at the current layer will be updated with the result of this method when called for the underlaying layer.

    Parameters

    • config: any

      The config at the current layer.

    • key: string

      The key of the property which should be updated/set. Must be given as path starting the root of the configuration.

    • value: any

      The new value of the property.

    Returns any

    Returns the updated config.
